```java
public record Person(String name, String email) {}

@Route("form")
public class FormView extends FormLayout {
  public FormView() {
    TextField name = new TextField("Name");
    EmailField email = new EmailField("Email");
    Label status = new Label();
    Button save = new Button("Save");

    Binder<Person> binder = new Binder<>(Person.class);
    binder.forField(name).asRequired("Name required").bind(Person::name, null);
    binder.forField(email).asRequired("Email required").bind(Person::email, null);
    binder.setBean(new Person("", ""));

    save.addClickListener(e -> {
      if (binder.validate().isOk())
        status.setText("âœ… Saved " + name.getValue());
    });

    add(new H2("Form Validation Demo"), name, email, save, status);
  }
}
```

## 7ï¸âƒ£ BCE Architecture â€” To-Do Demo

```
todo/
â”œâ”€ entity/Todo.java

â””â”€ boundary/TodoView.java
```

**Entity**

```java
public class Todo {
  private Long id;
  private String title;
  private boolean done;
  // getters & setters
}
```

**Control**

```java
@Service
public class TodoService {
  private final List<Todo> todos = new ArrayList<>();

  public List<Todo> findAll() { return todos; }
  public void add(String title) {
    Todo t = new Todo();
    t.setId((long)(todos.size()+1));
    t.setTitle(title);
    t.setDone(false);
    todos.add(t);
  }
  public void toggle(Long id) {
    todos.stream().filter(t -> t.getId().equals(id))
      .findFirst().ifPresent(t -> t.setDone(!t.isDone()));
  }
}
```

**Boundary**

```java
@Route("todo")
public class TodoView extends VerticalLayout {
  private final TodoService service;
  private final Grid<Todo> grid = new Grid<>(Todo.class, false);

  public TodoView(TodoService service) {
    this.service = service;
    TextField title = new TextField("New task");
    Button add = new Button("Add", e -> {
      service.add(title.getValue());
      title.clear();
      refresh();
    });

    grid.addColumn(Todo::getId).setHeader("ID");
    grid.addColumn(Todo::getTitle).setHeader("Title");
    grid.addComponentColumn(todo -> {
      Button toggle = new Button(todo.isDone() ? "Undo" : "Done");
      toggle.addClickListener(e -> { service.toggle(todo.getId()); refresh(); });
      return toggle;
    });
    grid.setAllRowsVisible(true);

    add(new H2("ğŸ“ To-Do Manager (BCE)"), title, add, grid);
    refresh();
  }

  private void refresh() { grid.setItems(service.findAll()); }
}
```
